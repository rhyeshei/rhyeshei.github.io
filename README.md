# Portfolio Site - Ryusei Takai
ポートフォリオサイトです。

## 概要
自身のプロフィール、スキル、および制作した作品（Works）を展示するためのシングルページWebサイトです。

## 技術スタック
- **Frontend**: HTML5, CSS3, JavaScript (jQuery)
- **Design**: Google Fonts (Lora, Castoro), Fontshare (Glacial Indifference), Adobe Fonts

## 主な機能
- **レスポンシブデザイン**: デバイスに応じたレイアウトの最適化。
- **スクロールアニメーション**: JavaScriptを用いたフェードイン・クリップ表示エフェクト。
- **SNS連携**: Instagram、GitHub、Emailへのダイレクトリンク。

## ディレクトリ構成
- `index.html`: サイトのメイン構造。
- `style.css`: 背景画像（haikei_portfolio.png）の設定やグリッドレイアウトによる作品表示。
- `script.js`: スクロール検知によるアニメーション制御。
- `img/`: サイトで使用する画像やアイコン類。
